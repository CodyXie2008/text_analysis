# 文本分析算法文档

## 📋 文档概述

本文档集合包含了文本分析模块中所有核心算法的详细说明，包括算法原理、实现细节、使用方法和优化建议。

## 📚 算法文档索引

### 核心算法文档

#### 1. 数据清洗算法
- **文档**: [数据清洗算法详解.md](./数据清洗算法详解.md)
- **模块**: `data_cleaning_optimized.py`
- **功能**: 垃圾评论检测、文本清洗、智能分词、时间标准化
- **特点**: 支持本地/API双模式分词，自动父子评论识别

#### 2. 从众心理分析算法

##### 2.1 时间从众心理分析
- **文档**: [从众心理时间分析算法详解.md](./从众心理时间分析算法详解.md)
- **模块**: `conformity_time_analyzer.py`
- **功能**: 分析评论时间从众心理特征
- **算法**: 自适应衰减因子、指数衰减模型、时间分类
- **应用**: 社交媒体行为分析、内容影响力评估

##### 2.2 情感从众心理分析
- **文档**: [情感从众心理分析算法详解.md](./情感从众心理分析算法详解.md)
- **模块**: `sentiment_conformity_analyzer.py`
- **功能**: 分析情感从众心理特征
- **算法**: 基于阿里云API的情感分析、情感差异计算、归一化处理
- **应用**: 情感传播分析、群体情感一致性评估

##### 2.3 相似度从众心理分析
- **文档**: [相似度从众心理分析算法详解.md](./相似度从众心理分析算法详解.md)
- **模块**: `similarity_conformity_analyzer.py`
- **功能**: 分析文本相似度从众心理特征
- **算法**: 基于阿里云API的文本向量化、余弦相似度计算、相似度分类
- **应用**: 内容模仿分析、文本相似性评估

##### 2.4 点赞从众心理分析
- **文档**: [点赞从众心理分析算法详解.md](./点赞从众心理分析算法详解.md)
- **模块**: `like_conformity_analyzer.py`
- **功能**: 分析点赞从众心理特征
- **算法**: 点赞数差异计算、归一化处理、从众心理分类
- **应用**: 互动行为分析、内容受欢迎程度评估

#### 2.5 综合分析模块
- **文档**: [综合分析算法详解.md](./综合分析算法详解.md)
- **模块**: `text_analysis_unified.py`
- **功能**: 统一入口点，集成所有从众心理分析模块
- **算法**: 多维度分数聚合、综合评分计算、结果整合
- **应用**: 完整的从众心理评估、一键分析报告生成

## 🔧 技术栈

### 核心依赖
- **Python 3.8+**: 主要开发语言
- **pandas**: 数据处理和分析
- **numpy**: 数值计算
- **scikit-learn**: 机器学习算法
- **jieba**: 中文分词
- **requests**: HTTP请求处理

### 外部API
- **阿里云NLP API**: 中文分词、情感分析、文本向量化
- **HMAC-SHA1**: API签名算法

### 算法库
- **MinMaxNormalizer**: 自定义归一化算法
- **StandardScaler**: 标准化算法
- **PCA**: 主成分分析
