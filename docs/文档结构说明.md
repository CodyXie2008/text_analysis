## 🚀 使用建议

### 按角色使用文档

**开发者**
1. 先阅读 `README.md` 了解项目概况
2. 查看 `数据清洗算法详解.md` 了解数据预处理
3. 查看 `从众心理时间分析算法详解.md` 了解核心算法
4. 参考 `集成工作流总结报告.md` 了解最佳实践

**研究人员**
1. 重点关注两个算法详解文档
2. 查看集成工作流报告了解实际效果
3. 参考文档中的数学公式和算法原理

**用户**
1. 先阅读主 `README.md` 了解基本用法
2. 查看集成工作流报告了解推荐流程
3. 根据需要深入查看相关算法文档

## 📈 维护原则

3. **版本同步**：文档版本与代码版本保持同步

1. **内容准确性**：确保文档内容与代码实现一致
2. **结构清晰**：保持文档结构清晰易读
